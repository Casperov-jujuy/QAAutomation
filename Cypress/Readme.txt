# Proyecto de Automatización con Cypress para xAcademy

Este repositorio contiene un proyecto de automatización desarrollado con **Cypress**, enfocado en las prácticas y objetivos de la academia **xAcademy Automation**.

---

## Descripción del Proyecto

El objetivo principal de este proyecto es demostrar la implementación de pruebas automatizadas utilizando el framework Cypress para la página web:

[https://endearing-lollipop-19f16d.netlify.app/](https://endearing-lollipop-19f16d.netlify.app/)

---

## Tecnologías Utilizadas

* **Cypress**: Framework de pruebas de extremo a extremo.
* **JavaScript**: Lenguaje de programación.
* **Node.js**: Entorno de ejecución para JavaScript.
* **Git/GitHub**: Control de versiones y hospedaje del repositorio.

---

## Configuración y Ejecución

Para trabajar con este proyecto, asegúrate de tener **Node.js** instalado. Una vez que tengas el repositorio en tu máquina (al descargarlo o hacer un "fork" y clonarlo), sigue estos pasos:

1.  **Instala las dependencias**:
    Abre tu terminal en la carpeta raíz del proyecto y ejecuta:
    ```bash
    npm install
    ```
2.  **Ejecuta las pruebas**:
    Puedes ejecutar las pruebas en modo interactivo (abriendo el Test Runner de Cypress):
    ```bash
    npx cypress open
    ```
    O en modo headless (ejecutando las pruebas desde la línea de comandos sin interfaz gráfica):
    ```bash
    npx cypress run
    ```
